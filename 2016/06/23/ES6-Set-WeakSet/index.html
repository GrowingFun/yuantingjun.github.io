ES6-Set-WeakSet

ECMAScript 6 新增了下面四种数据结构：Map, WeakMap, Set and WeakSet.

Set

  1. Set类似于数组，但是成员的值都是唯一的，没有重复的值。

  2. 遍历操作

  3. Set结构转为数组结构，比较快速的方法是结合使用扩展运算符（…）,或者 Array.from()。

WeakSet

WeakSet和Set一样都不存储重复的元素，但有一些不同点 。
WeakSet的成员只能是对象，而不能是其他类型的值。否则报错：Uncaught TypeError: Invalid value used in weak set
WeakSet没有size属性，没有办法遍历它的成员。
