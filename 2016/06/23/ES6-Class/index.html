ES6-Class

通过class关键字，可以定义类。

  1. constructor方法
    constructor方法是类的默认方法，通过new命令生成对象实例时，自动调用该方法。

  2. name属性
    class Point {}
    Point.name // “Point”

  3. class表达式
    与函数一样，Class也可以使用表达式的形式

const MyClass = class Me {
getClassName() {
return Me.name;
}
};

  1. Class的继承
    Class之间可以通过extends关键字，实现继承，这比ES5的通过修改原型链实现继承，要清晰和方便很多。
    子类会继承父类的属性和方法。
    当类定义很多时，在webstrom 中可以通过F4查看继承关系
  1. class的取值函数（getter）和存值函数（setter）

在Class内部可以使用get和set关键字，对某个属性设置存值函数和取值函数。
getter限定了函数只能取值，

setter函数限定函数只能赋值

  1. Class的Generator方法
    如果某个方法之前加上星号（*），就表示该方法是一个Generator函数。

  2. Class的静态方法

注意：

  1. 使用class 继承时，必须在构造函数中调用super方法，而且super方法必须其他赋值语句之前。
